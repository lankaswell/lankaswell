THE LANKASWELL PROJECT

Hey there!

LankaSwell is an independent, self-funded project building free live surf cameras across Sri Lanka. No company, no investors, just a simple idea: make the ocean more visible before you paddle out.

LankaSwell is free to use and aims to remain that way. Donations simply help keep the cameras online and allow new spots to be added along the coast.

Do you want to participate or collaborate to LankaSwell? See how below!

Curious about the philosophy behind the project, privacy, or how LankaSwell works? Read the Principles page.
